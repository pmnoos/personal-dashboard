# Personal Dashboard

A lightweight personal dashboard built with plain HTML, CSS, and JavaScript.

## Live Demo

- [Open Personal Dashboard](https://pmnoos.github.io/personal-dashboard/)

## Features

- Weather widget with forecast toggle
- World clock widget (Local, London, Brisbane)
- GBP ↔ AUD currency converter with live rate
- Calendar with events modal
- To-do list with local storage
- BBC Sounds widget with VPN shortcuts
- Australia radio quick-launch widget
- Responsive layout for desktop, tablet, and mobile

## Project Structure

- `index.html` — dashboard markup
- `style.css` — styling and responsive layout
- `js/weather.js` — weather logic
- `js/world-clock.js` — world clock logic
- `js/currency.js` — currency conversion logic
- `js/calendar.js` — calendar logic
- `js/todo.js` — to-do logic
- `js/podcasts.js` — BBC sounds widget logic
- `js/australia-radio.js` — Australia radio widget logic

## Run locally

You can open `index.html` directly, or run with a local static server (recommended).

## Notes

This project uses browser local storage for tasks/events and links out to third-party radio/player services.
